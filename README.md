# monthly-vol7

## development

### install dependencies

```bash
pnpm i
```

### create environment files

```bash
cp apps/frontend/.env.local.example apps/frontend/.env.local
cp apps/backend/.dev.vars.example apps/backend/.dev.vars
```

### run dev server

```bash
pnpm dev
```

### check format and lint

```bash
pnpm check
```

### format and lint fix

```bash
pnpm fix
```

### typecheck

```bash
pnpm typecheck
```
